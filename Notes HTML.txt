                                                HTML Terminology

HTML Element - a element displayed on a web-page (exp. button)
Syntax - rules for writting code (exp. <></>)
HTML Attribute - modifies how a element behaves (exp. <a href="">) *Every HTML element has its own attributes
Indet - in TAB
HTML Entity - object in text (exp. · , ✓)
Text Element - element that is incide a line of text
Nesting - element inside other element (exp. <html><head></head></html>)
Void Elements - don't need a closing tag
Filepaths - hrev="folder1/folder2/file.code"


                                                HTML Elements
<p> - paragraph   *by default, come with margin at the top and bottom
<a> - anchor element (link to another website)
    attributes:
        href="" - for url website
        target="" - purpose (exp. open an new web-page - target="_blank")
<button> - button
    attributes:
        class="" - give a special name for button
<Style> - for change HTML with the help CSS
<img> - for image
    attributes:
        src="" 
    specific css:
        width and height - 1/2 image save shape; 2/2 image deform
        object-fit: cover; - image cut to the parameters (width and height)
            + object-position: right, left;  *defolt - center
        object-fit: contain; - reduces the image to parameters, leaving empty fields
            + object-position: top, bottom;  *defolt - center
<input>
    attribute:
        type="text" - line to fill in
        type="checkbox" - checkbox
        placeholder="" - exp. for window search

<br> - line break - void elements
